# 小红书 AI 教程口播脚本生成器

一个实用的口播脚本写作工具，帮你快速生成结构清晰的教程视频脚本。

## 📌 这是什么

这是一个**经过爆款视频验证过的可复用 Skills 技能**。通过分析大量高互动教程视频，总结出一套行之有效的脚本结构。

**核心理念**：好的脚本有规律可循，新手可以直接套用。

**已验证效果**：该结构已帮助多个新手博主产出高互动内容，单条视频收藏量 1000+。

## ✨ 功能特点

- **结构化框架** - 6 段式脚本结构，每段作用清晰
- **开箱即用** - 内置 3 个完整示例，改改用就行
- **标题公式库** - 解决"起标题困难"
- **多场景适用** - AI 工具教程、软件教学、技能分享都可用

## 🎯 适合谁用

- 刚开始做小红书/抖音的新手博主
- 需要快速生产教程内容的创作者
- 想学习口播脚本结构的人

## 📦 包含内容

```
xiaohongshu-script-generator/
├── SKILL.md                  # 详细使用说明
├── README.md                 # 本文件
└── references/
    ├── prompt-template.md    # Prompt 模板（复制即用）
    ├── examples.md           # 3 个脚本示例
    └── title-formulas.md     # 标题公式和词库
```

## 🚀 快速开始

### 方式 1：使用 AI 助手生成

如果你使用支持 Skills 的 AI 助手，直接说：

> "帮我写一个口播脚本，工具是即梦 AI，功能是照片转油画风格"

### 方式 2：复制 Prompt 模板

1. 打开 `references/prompt-template.md`
2. 复制完整 Prompt
3. 填写你的信息
4. 发送给任意大模型

### 方式 3：参考示例修改

1. 打开 `references/examples.md`
2. 选择一个最接近的示例
3. 替换成你的内容

## 📝 使用示例

**输入信息**：
```
- AI 工具名称：即梦 AI
- 核心功能：照片转油画风格
- 目标用户：想制作艺术头像的普通人
- 最终产出：梵高风格头像
- 具体案例：用自拍照变成星空风格
```

**输出包含**：
- 标题（20 字内）
- 封面文案（主标题 + 副标题）
- 口播脚本（300-400 字）
- 提示词模板
- 评论区引导（3 条）

## 💡 使用建议

1. **先模仿再创新** - 先用示例模板，熟练后再调整
2. **保持口语化** - 写完后读一遍，不顺口的地方改掉
3. **案例要具体** - "林黛玉初见贾宝玉"比"古典故事"更好
4. **测试迭代** - 发出去看数据，好的结构保留

## ⚠️ 注意事项

- 这个工具提供的是**结构**，内容需要你自己填充
- 不同平台可能需要调整风格（抖音更快、B 站更详细）
- 提示词模板需要根据实际工具调整
- 建议配合实际测试优化脚本
