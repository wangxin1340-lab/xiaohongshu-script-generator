# 标题公式库

常用的标题写法和情绪词库，解决"起标题困难"。

**使用建议**：
- 直接套用公式，替换括号里的内容
- 从情绪词库选一个合适的词
- 写完后检查是否超过 20 字

---

## 数字 + 效果 + 情绪词

```
[时间] 把 [输入] 变成 [输出]，[情绪词]！
```

示例：
- 30 秒把课本变成皮影戏动画，建议收藏！
- 1 分钟把自拍照变成宫崎骏女主，太绝了！
- 30 秒用 AI 生成专业 Logo，省了 5000 块！
- 10 秒把文字变成 3D 模型，真香！

## 问题 + 解决方案

```
如何 [达成目标]？[时间] 用 [工具] 就能搞定
```

示例：
- 如何做出电影感视频？30 秒用即梦就能搞定
- 如何设计高级感海报？1 分钟用 AI 就能完成
- 如何生成专属头像？免费 AI 就能搞定

## 对比 + 价值

```
[过去方式]vs[现在方式]，[时间] 搞定 [成果]
```

示例：
- 设计师收费 5000vsAI 免费，30 秒搞定 Logo
- 学习 3 个月 vsAI30 秒，皮影戏动画轻松拿捏
- 手动 P 图 1 小时 vsAI1 分钟，动漫头像自由

## 人群定向 + 痛点

```
[目标人群] 必看！[时间] 解决 [痛点]
```

示例：
- 老师必看！30 秒把课本变成动画微课
- 自媒体人必看！1 分钟生成 100 个选题
- 设计师必看！AI 帮你搞定初稿，效率翻倍

## 情绪词库

**强烈推荐类：**
- 建议收藏！
- 一定要看！
- 千万别错过！
- 赶紧码住！

**惊叹类：**
- 太绝了！
- 真香！
- 惊呆了！
- 绝绝子！

**价值类：**
- 省了 5000 块！
- 效率翻倍！
- 零基础也能行！
- 看完你也能搞定！

## 封面文案公式

```
主标题：[核心动作]（4-6 字）
副标题：[时间/价值]（4-6 字）
```

示例：
- 主：课本变皮影 | 副：30 秒搞定
- 主：照片变动漫 | 副：宫崎骏风格
- 主：AI 生成 Logo | 副：省 5000 块
- 主：文字变 3D | 副：1 分钟出图

## 避坑指南

❌ 避免：
- 超过 20 字
- 专业术语
- 模糊表述（如"很好用"）
- 纯工具名（如"即梦 AI 教程"）

✅ 推荐：
- 具体数字（30 秒、1 分钟）
- 具体效果（皮影戏、宫崎骏风格）
- 具体价值（省 5000 块、效率翻倍）
- 情绪词（太绝了、建议收藏）
